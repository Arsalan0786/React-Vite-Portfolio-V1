‚öôÔ∏è Local Development Setup
Follow these steps to get a local copy of the project running on your machine for development and testing.

Prerequisites

You need to have Node.js and npm (or yarn / pnpm) installed.

Installation

Clone the repository:

Bash
git clone [https://github.com/Arsalan0786/React-Vite-Portfolio-V1]
cd [React-Vite-Portfolio-V1]
Install dependencies:

Bash
npm install  # or yarn install / pnpm install
Run the development server:

Bash
npm run dev  # or yarn dev / pnpm dev
The application will now be running locally, usually accessible at http://localhost:5173.

Production Build

To create the optimized, final static files:

Bash
npm run build
The optimized files will be generated in the build directory (which was configured in Vercel settings).
